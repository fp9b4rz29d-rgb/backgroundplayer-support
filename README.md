  ## プライバシーポリシー

  BackgroundPlayer（以下「本アプリ」）は、以下のプライバシーポリシーを定めます。

  ### 1. 収集する情報
  本アプリは、ユーザーの個人情報を収集しません。動画ファイルはすべてお使いのデバイス内にのみ保存され、外部サーバーに送信されることはありません。

  ### 2. 広告について
  本アプリはGoogle AdMobを使用して広告を配信しています。AdMobは広告配信のために匿名の識別子を使用する場合があります。詳細はGoogleのプライバシーポリシーをご確認ください。
  https://policies.google.com/privacy

  ### 3. 写真ライブラリへのアクセス
  本アプリは動画を読み込むために写真ライブラリへのアクセス許可を求めます。これはユーザーが選択した動画を読み込む目的でのみ使用され、データが外部に送信されることはありません。

  ### 4. 変更について
  本ポリシーは予告なく変更されることがあります。
